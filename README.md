# tulingRobot-wechat-applet

微信小程序练手项目 - 笑话大全。每小时更新。笑话大全，信息搜集整理于互联网，包括文字笑话、搞笑图片等。

小程序所用的数据接口API来源：[易源数据 - 笑话大全](https://www.showapi.com/api/view/341)
